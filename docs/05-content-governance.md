# 内容治理与脱敏规范

## 1. 目标

本规范用于确保全球周期投资模拟器保持教育模拟定位，避免被误解为真实证券推荐或实盘投资辅助工具。

## 2. 核心原则

- 所有用户可见资产均为教学抽象原型。
- 不展示真实单一证券名称、ticker、logo 或可唯一识别公司的细节组合。
- 不接入真实行情数据。
- 不输出买入、卖出、持有任何真实证券的建议。
- 允许使用真实市场分类和行业分类，因为它们是教学所需的宏观概念。

## 3. 允许内容

允许用户可见：

- 市场分类：A股、H股、美股、日股、台股、韩股。
- 行业分类：消费、科技、医药、新能源、有色、金融、军工、农业。
- 宏观概念：复苏、过热、滞胀、衰退、宽松、紧缩。
- 抽象资产风格：北美算力平台股、东亚先进制造股、高股息金融股、防御消费龙头股。
- 脱敏后的近似名称。

## 4. 禁止内容

禁止用户可见：

- 真实公司全称。
- 真实股票简称。
- ticker 或证券代码。
- 公司 logo。
- 真实历史 K 线截图。
- 可唯一识别某公司的业务、财务、创始人、产品组合描述。
- 对真实证券的买卖建议。
- 暗示系统可以预测真实市场收益的文案。

## 5. 脱敏命名规则

如果必须使用接近真实风格的教学名称，采用以下规则：

1. 保留原名称的行业、地区或语感风格。
2. 改动至少一个关键字，使其不再是真实证券简称。
3. 不同时展示真实 ticker、logo、业务细节。
4. 不在前端可见数据中保存“真实原型 → 脱敏名称”的映射。
5. 每个脱敏名称只作为教学虚构资产，不暗示可投资。

示例：

| 真实名称 | 脱敏名称 |
| --- | --- |
| 英伟达 | 英伟大 |
| 苹果 | 苹国 |
| 特斯拉 | 特思拉 |
| 台积电 | 台晶电 |
| 三星电子 | 三新电子 |
| 腾讯控股 | 腾迅控股 |
| 阿里巴巴 | 阿理巴巴 |
| 贵州茅台 | 贵州茅坛 |
| 宁德时代 | 宁得时代 |
| 比亚迪 | 比雅迪 |

## 6. MVP 内容策略

MVP 默认不展示股票级资产。页面只展示：

- 年度宏观环境。
- 市场配置。
- 板块配置。
- 事件卡。
- 组合结果。
- 复盘解释。

如需增强代入感，可使用抽象资产风格标签，例如：

- 北美科技成长资产。
- 东亚半导体周期资产。
- 港股低估值平台资产。
- A股政策成长资产。
- 日股防御出口资产。
- 韩股周期制造资产。

## 7. 免责声明

以下文案必须在首页或页脚可见：

> 本产品为教育模拟工具，所有资产名称均为教学抽象原型，不构成任何真实证券投资建议。

在结算页和结局页可使用短版：

> 模拟结果仅用于学习周期、配置与风控逻辑，不代表真实市场表现。

## 8. 内容审查清单

每次新增内容前检查：

- 是否出现真实公司名？
- 是否出现 ticker 或证券代码？
- 是否出现 logo 或真实截图？
- 是否可通过描述唯一识别某公司？
- 是否像是在建议买卖某真实证券？
- 是否让用户误以为模拟收益可复制到真实市场？

任一答案为“是”时，必须修改或删除该内容。
